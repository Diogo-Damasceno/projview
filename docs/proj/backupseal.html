# backupseal Verifica integridade de backups por hash antes de restaurar. ## O que faz Esta ferramenta de **defensiva** contra alvos que voce possui ou tem autorizacao. Verifica integridade de backups por hash antes de restaurar. O codigo e testavel localmente (sem dependencias de rede para os testes unitarios). > **Aviso etico:** uso exclusivo para fins educacionais e testes em ambientes > que voce **possui** ou tem **autorizacao expressa** para auditar. Nunca a > utilize contra sistemas de terceiros. O autor nao se responsabiliza por uso > indevido. ## Instalacao ```bash git clone https://github.com/Diogo-Damasceno/backupseal.git cd backupseal python3 -m venv .venv && . .venv/bin/activate pip install -e . ``` ## Uso Confirme que seus backups não foram corrompidos. ## Exemplos reais ```bash $ backup-verifier backup1.tar backup2.tar # {'backup1.tar': 'e5f6...'} ``` ## Como funciona (resumo) O modulo principal implementa a logica em funcoes puras e testaveis; a CLI e apenas a casca de argumentos. Os testes em `tests/` (Python) ou `CoreTest.java` (Java) cobrem os casos principais e rodam offline. ## O que NAO faz - Nao executa ataques contra terceiros. - Nao exfiltra dados. - Nao substitui uma solucao comercial de seguranca. ## Licenca MIT — ver `LICENSE`.

Como rodar na prática (entrada → saída):

$ backupseal verificar backup1.tar backup2.tar
backup1.tar OK (sha256 confere)\nbackup2.tar CORROMPIDO
