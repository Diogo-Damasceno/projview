# redirect-find Classifica redirecionamentos abertos por pontuação de risco. > **Aviso etico:** ferramenta exclusiva para fins educacionais e testes em > ambientes que voce possui ou tem autorizacao para auditar. Categoria: > Ofensiva. Nunca a use contra terceiros. ## Instalacao ```bash git clone https://github.com/Diogo-Damasceno/redirect-find.git cd redirect-find python3 -m venv .venv && . .venv/bin/activate pip install -e . ``` ## Uso Encontre open-redirect em seus próprios apps. Exemplos reais: ``` $ redirect-find 'http://a.com//evil' 80 (alto)``` ## Arquitetura `src/redirect_find/core.py` tem a logica pura (funcoes testaveis); `cli.py` e a casca de argumentos. Testes em `tests/` cobrem os casos principais.
