# java-filehash Hash SHA-256 de arquivos em Java (integridade). > **Aviso etico:** ferramenta exclusiva para fins educacionais e testes em > ambientes que voce possui ou tem autorizacao para auditar. Categoria: > Defensiva. Nunca a use contra terceiros. ## Instalacao ```bash git clone https://github.com/Diogo-Damasceno/java-filehash.git cd java-filehash python3 -m venv .venv && . .venv/bin/activate pip install -e . ``` ## Uso Verifique integridade de backups seus. Exemplos reais: ``` $ java filehash.Main arquivo.txt <hash>``` ## Arquitetura `src/filehash/core.py` tem a logica pura (funcoes testaveis); `cli.py` e a casca de argumentos. Testes em `tests/` cobrem os casos principais.
