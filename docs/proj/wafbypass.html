# wafbypass Classifica vetores de evasão de WAF por pontuação de risco. ## O que faz Esta ferramenta de **ofensiva** apenas em aplicacoes suas ou engajamento autorizado. Classifica vetores de evasão de WAF por pontuação de risco. O codigo e testavel localmente (sem dependencias de rede para os testes unitarios). > **Aviso etico:** uso exclusivo para fins educacionais e testes em ambientes > que voce **possui** ou tem **autorizacao expressa** para auditar. Nunca a > utilize contra sistemas de terceiros. O autor nao se responsabiliza por uso > indevido. ## Instalacao ```bash git clone https://github.com/Diogo-Damasceno/wafbypass.git cd wafbypass python3 -m venv .venv && . .venv/bin/activate pip install -e . ``` ## Uso Envie variações ao seu próprio WAF e veja o que passa. ## Exemplos reais ```bash $ waf-tester '<ScRiPt>alert(1)</sCriPt>' # 95 (alto) ``` ## Como funciona (resumo) O modulo principal implementa a logica em funcoes puras e testaveis; a CLI e apenas a casca de argumentos. Os testes em `tests/` (Python) ou `CoreTest.java` (Java) cobrem os casos principais e rodam offline. ## O que NAO faz - Nao executa ataques contra terceiros. - Nao exfiltra dados. - Nao substitui uma solucao comercial de seguranca. ## Licenca MIT — ver `LICENSE`.
