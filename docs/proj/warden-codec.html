# warden-codec Codifica/decodifica payloads em base64/hex/url para análise segura. ## O que faz Esta ferramenta de **defensiva** contra alvos que voce possui ou tem autorizacao. Codifica/decodifica payloads em base64/hex/url para análise segura. O codigo e testavel localmente (sem dependencias de rede para os testes unitarios). > **Aviso etico:** uso exclusivo para fins educacionais e testes em ambientes > que voce **possui** ou tem **autorizacao expressa** para auditar. Nunca a > utilize contra sistemas de terceiros. O autor nao se responsabiliza por uso > indevido. ## Instalacao ```bash git clone https://github.com/Diogo-Damasceno/warden-codec.git cd warden-codec python3 -m venv .venv && . .venv/bin/activate pip install -e . ``` ## Uso Codifique um payload antes de testar WAF/input-validation em apps suas. ## Exemplos reais ```bash $ payload-coder encode base64 '<?php eval($_GET[x]);?>' # PD9waHAgZXZhbCgkX0dFVFt4XSk7Pz4= ``` ```bash $ payload-coder decode hex 414243 # ABC ``` ## Como funciona (resumo) O modulo principal implementa a logica em funcoes puras e testaveis; a CLI e apenas a casca de argumentos. Os testes em `tests/` (Python) ou `CoreTest.java` (Java) cobrem os casos principais e rodam offline. ## O que NAO faz - Nao executa ataques contra terceiros. - Nao exfiltra dados. - Nao substitui uma solucao comercial de seguranca. ## Licenca MIT — ver `LICENSE`.

Como rodar na prática (entrada → saída):

$ warden-codec encode base64 '<?php eval($_GET[x]);?>'
PD89cGhwIGV2YWwoJF9HRVRbeF0pOz8+
$ warden-codec decode hex 4142
AB
