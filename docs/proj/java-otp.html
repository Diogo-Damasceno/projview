# java-otp Gerador TOTP/HOTP simplificado em Java (base32). > **Aviso etico:** ferramenta exclusiva para fins educacionais e testes em > ambientes que voce possui ou tem autorizacao para auditar. Categoria: > Segurança. Nunca a use contra terceiros. ## Instalacao ```bash git clone https://github.com/Diogo-Damasceno/java-otp.git cd java-otp python3 -m venv .venv && . .venv/bin/activate pip install -e . ``` ## Uso Estude 2FA em laboratório. Exemplos reais: ``` $ java otptool.Main seed <codigo>``` ## Arquitetura `src/otptool/core.py` tem a logica pura (funcoes testaveis); `cli.py` e a casca de argumentos. Testes em `tests/` cobrem os casos principais.
