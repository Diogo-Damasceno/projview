# xss-forge Forja/ofusca payloads XSS para testes de WAF/input-validation. > **Aviso etico:** ferramenta exclusiva para fins educacionais e testes em > ambientes que voce possui ou tem autorizacao para auditar. Categoria: > Ofensiva. Nunca a use contra terceiros. ## Instalacao ```bash git clone https://github.com/Diogo-Damasceno/xss-forge.git cd xss-forge python3 -m venv .venv && . .venv/bin/activate pip install -e . ``` ## Uso Teste seus próprios apps contra filtros de XSS (autorizado). Exemplos reais: ``` $ xss-forge encode url '<script>alert(1)</script>' %3Cscript%3E...``` ## Arquitetura `src/xss_forge/core.py` tem a logica pura (funcoes testaveis); `cli.py` e a casca de argumentos. Testes em `tests/` cobrem os casos principais.
