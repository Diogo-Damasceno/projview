# reconmap Reconhecimento de footprint de seu próprio domínio (DNS). ## O que faz Esta ferramenta de **ofensiva** apenas em aplicacoes suas ou engajamento autorizado. Reconhecimento de footprint de seu próprio domínio (DNS). O codigo e testavel localmente (sem dependencias de rede para os testes unitarios). > **Aviso etico:** uso exclusivo para fins educacionais e testes em ambientes > que voce **possui** ou tem **autorizacao expressa** para auditar. Nunca a > utilize contra sistemas de terceiros. O autor nao se responsabiliza por uso > indevido. ## Instalacao ```bash git clone https://github.com/Diogo-Damasceno/reconmap.git cd reconmap python3 -m venv .venv && . .venv/bin/activate pip install -e . ``` ## Uso Mapa de superfície para um engajamento autorizado. ## Exemplos reais ```bash $ recon-foot meu.com www,mail # www.meu.com -> 1.1.1.1 ``` ## Como funciona (resumo) O modulo principal implementa a logica em funcoes puras e testaveis; a CLI e apenas a casca de argumentos. Os testes em `tests/` (Python) ou `CoreTest.java` (Java) cobrem os casos principais e rodam offline. ## O que NAO faz - Nao executa ataques contra terceiros. - Nao exfiltra dados. - Nao substitui uma solucao comercial de seguranca. ## Licenca MIT — ver `LICENSE`.

Como rodar na prática (entrada → saída):

$ reconmap exemplo.com
hosts: 3 | portas abertas: 22,80,443 | exposicao: 1 (80 sem HSTS)
