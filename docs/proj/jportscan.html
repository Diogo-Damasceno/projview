# jportscan Scanner de portas TCP em Java (contra alvos próprios). ## O que faz Esta ferramenta de **defensiva** contra alvos que voce possui ou tem autorizacao. Scanner de portas TCP em Java (contra alvos próprios). O codigo e testavel localmente (sem dependencias de rede para os testes unitarios). > **Aviso etico:** uso exclusivo para fins educacionais e testes em ambientes > que voce **possui** ou tem **autorizacao expressa** para auditar. Nunca a > utilize contra sistemas de terceiros. O autor nao se responsabiliza por uso > indevido. ## Instalacao ```bash git clone https://github.com/Diogo-Damasceno/jportscan.git cd jportscan # Java 21 make build ``` ## Uso Mapeie portas da sua infraestrutura. ## Exemplos reais ```bash $ java portscan.Main 127.0.0.1 22 # aberta/fechada ``` ## Como funciona (resumo) O modulo principal implementa a logica em funcoes puras e testaveis; a CLI e apenas a casca de argumentos. Os testes em `tests/` (Python) ou `CoreTest.java` (Java) cobrem os casos principais e rodam offline. ## O que NAO faz - Nao executa ataques contra terceiros. - Nao exfiltra dados. - Nao substitui uma solucao comercial de seguranca. ## Licenca MIT — ver `LICENSE`.

$ java jportscan.Main 127.0.0.1 20-25
