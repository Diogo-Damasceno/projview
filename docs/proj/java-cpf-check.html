README

# java-cpf-check Validador de CPF (dígitos verificadores) em Java. > **Aviso etico:** ferramenta exclusiva para fins educacionais e testes em > ambientes que voce possui ou tem autorizacao para auditar. Categoria: > Defensiva. Nunca a use contra terceiros. ## Instalacao ```bash git clone https://github.com/Diogo-Damasceno/java-cpf-check.git cd java-cpf-check python3 -m venv .venv && . .venv/bin/activate pip install -e . ``` ## Uso Valide CPF em seus sistemas. Exemplos reais: ``` $ java cpfcheck.Main cpf 12345678909 valido``` ## Arquitetura `src/cpfcheck/core.py` tem a logica pura (funcoes testaveis); `cli.py` e a casca de argumentos. Testes em `tests/` cobrem os casos principais.
