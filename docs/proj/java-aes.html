# java-aes Cifra/decifra AES (demonstração, autorizado). > **Aviso etico:** ferramenta exclusiva para fins educacionais e testes em > ambientes que voce possui ou tem autorizacao para auditar. Categoria: > Privacidade. Nunca a use contra terceiros. ## Instalacao ```bash git clone https://github.com/Diogo-Damasceno/java-aes.git cd java-aes python3 -m venv .venv && . .venv/bin/activate pip install -e . ``` ## Uso Teste cifragem simétrica em laboratório. Exemplos reais: ``` $ java aestool.Main enc chave segredo <base64>``` ## Arquitetura `src/aestool/core.py` tem a logica pura (funcoes testaveis); `cli.py` e a casca de argumentos. Testes em `tests/` cobrem os casos principais.
