O autor nao se responsabiliza por uso > indevido. ## Instalacao ```bash git clone https://github.com/Diogo-Damasceno/leakheaders.git cd leakheaders python3 -m venv .venv && . .venv/bin/activate pip install -e . ``` ## Uso Identifique fingerprinting em suas respostas. ## Exemplos reais ```bash $ header-leak headers.txt # server-leak: 1 ``` ## Como funciona (resumo) O modulo principal implementa a logica em funcoes puras e testaveis; a CLI e apenas a casca de argumentos.
