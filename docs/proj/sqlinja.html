# sqlinja Forja/ofusca payloads SQLi para testes de input-validation. ## O que faz Esta ferramenta de **ofensiva** apenas em aplicacoes suas ou engajamento autorizado. Forja/ofusca payloads SQLi para testes de input-validation. O codigo e testavel localmente (sem dependencias de rede para os testes unitarios). > **Aviso etico:** uso exclusivo para fins educacionais e testes em ambientes > que voce **possui** ou tem **autorizacao expressa** para auditar. Nunca a > utilize contra sistemas de terceiros. O autor nao se responsabiliza por uso > indevido. ## Instalacao ```bash git clone https://github.com/Diogo-Damasceno/sqlinja.git cd sqlinja python3 -m venv .venv && . .venv/bin/activate pip install -e . ``` ## Uso Valide defesas SQLi em aplicações suas (autorizado). ## Exemplos reais ```bash $ sqli-forge encode hex "' OR '1'='1" # 27204f523131... ``` ## Como funciona (resumo) O modulo principal implementa a logica em funcoes puras e testaveis; a CLI e apenas a casca de argumentos. Os testes em `tests/` (Python) ou `CoreTest.java` (Java) cobrem os casos principais e rodam offline. ## O que NAO faz - Nao executa ataques contra terceiros. - Nao exfiltra dados. - Nao substitui uma solucao comercial de seguranca. ## Licenca MIT — ver `LICENSE`.
