O autor nao se responsabiliza por uso > indevido. ## Instalacao ```bash git clone https://github.com/Diogo-Damasceno/jhashcrack.git cd jhashcrack # Java 21 make build ``` ## Uso Gere hashes para seus próprios desafios. ## Exemplos reais ```bash $ java hashcrack.Main 1234 # <md5> ``` ## Como funciona (resumo) O modulo principal implementa a logica em funcoes puras e testaveis; a CLI e apenas a casca de argumentos.
