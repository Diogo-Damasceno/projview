# portscope Scanner de portas TCP (contra alvos que você possui). ## O que faz Esta ferramenta de **defensiva** contra alvos que voce possui ou tem autorizacao. Scanner de portas TCP (contra alvos que você possui). O codigo e testavel localmente (sem dependencias de rede para os testes unitarios). > **Aviso etico:** uso exclusivo para fins educacionais e testes em ambientes > que voce **possui** ou tem **autorizacao expressa** para auditar. Nunca a > utilize contra sistemas de terceiros. O autor nao se responsabiliza por uso > indevido. ## Instalacao ```bash git clone https://github.com/Diogo-Damasceno/portscope.git cd portscope python3 -m venv .venv && . .venv/bin/activate pip install -e . ``` ## Uso Mapeie serviços expostos em sua própria infraestrutura. ## Exemplos reais ```bash $ port-lister 127.0.0.1 22,80,443 # abertas: [22, 80] ``` ## Como funciona (resumo) O modulo principal implementa a logica em funcoes puras e testaveis; a CLI e apenas a casca de argumentos. Os testes em `tests/` (Python) ou `CoreTest.java` (Java) cobrem os casos principais e rodam offline. ## O que NAO faz - Nao executa ataques contra terceiros. - Nao exfiltra dados. - Nao substitui uma solucao comercial de seguranca. ## Licenca MIT — ver `LICENSE`.
