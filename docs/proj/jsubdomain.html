# jsubdomain Resolver de subdomínios em Java para recon próprio. ## O que faz Esta ferramenta de **ofensiva** apenas em aplicacoes suas ou engajamento autorizado. Resolver de subdomínios em Java para recon próprio. O codigo e testavel localmente (sem dependencias de rede para os testes unitarios). > **Aviso etico:** uso exclusivo para fins educacionais e testes em ambientes > que voce **possui** ou tem **autorizacao expressa** para auditar. Nunca a > utilize contra sistemas de terceiros. O autor nao se responsabiliza por uso > indevido. ## Instalacao ```bash git clone https://github.com/Diogo-Damasceno/jsubdomain.git cd jsubdomain # Java 21 make build ``` ## Uso Recon pasivo do seu domínio. ## Exemplos reais ```bash $ java subdomain.Main www.exemplo.com # 1.2.3.4 ``` ## Como funciona (resumo) O modulo principal implementa a logica em funcoes puras e testaveis; a CLI e apenas a casca de argumentos. Os testes em `tests/` (Python) ou `CoreTest.java` (Java) cobrem os casos principais e rodam offline. ## O que NAO faz - Nao executa ataques contra terceiros. - Nao exfiltra dados. - Nao substitui uma solucao comercial de seguranca. ## Licenca MIT — ver `LICENSE`.

Como rodar na prática (entrada → saída):

$ java jsubdomain.Main exemplo.com www,api,dev
www -> 192.0.2.10\napi -> 192.0.2.20\ndev -> NXDOMAIN
