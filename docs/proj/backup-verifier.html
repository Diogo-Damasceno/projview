README

# backup-verifier Verifica integridade de backups por hash antes de restaurar. > **Aviso etico:** ferramenta exclusiva para fins educacionais e testes em > ambientes que voce possui ou tem autorizacao para auditar. Categoria: > Defensiva. Nunca a use contra terceiros. ## Instalacao ```bash git clone https://github.com/Diogo-Damasceno/backup-verifier.git cd backup-verifier python3 -m venv .venv && . .venv/bin/activate pip install -e . ``` ## Uso Confirme que seus backups não foram corrompidos. Exemplos reais: ``` $ backup-verifier backup1.tar backup2.tar {'backup1.tar': 'e5f6...'}``` ## Arquitetura `src/backup_verifier/core.py` tem a logica pura (funcoes testaveis); `cli.py` e a casca de argumentos. Testes em `tests/` cobrem os casos principais.
