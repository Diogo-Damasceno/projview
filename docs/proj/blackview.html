README

# blackview TUI interativa para o **BlackArch Linux** — lista, instala e executa ferramentas de segurança, no estilo do **dockview**. ## Funcionalidades - **Tools**: lista as ferramentas do BlackArch (sync do `pacman -Sl blackarch` ou cache de fallback embutido). Filtra por nome/descrição/categoria. - **Categorias**: agrupa por categoria (scanner, cracker, sniffer, forensic, malware, wireless, web, defensive, ...). - **Installed**: mostra o que já está instalado no sistema. - **Log**: saída das ações (instalação, --help das ferramentas). ## Atalhos | Tecla | Ação | |-------|------| | `tab` / `shift+tab` | trocar aba | | `r` | atualizar lista | | `f` | filtrar | | `i` | instalar ferramenta selecionada (sudo) | | `u` | remover ferramenta | | `x` | ver saída de `--help` da ferramenta | | `t` | abrir ferramenta num terminal gráfico (kitty/foot/alacritty) | | `s` | sincronizar repo BlackArch | | `q` | sair | ## Instalação ```bash python3 -m venv .venv && . .venv/bin/activate pip install -e . ``` Dependências: `textual>=8.2`. Para instalar ferramentas de verdade, precisa do repo **BlackArch** habilitado e de `sudo`: ```bash curl -s https://blackarch.org/strap.sh | sudo sh sudo pacman -Syu ``` Sem o repo, a TUI ainda abre com o cache de fallback (lista representativa). ## ⚠️ Aviso ético O BlackArch reúne ferramentas de **ofensivas e defensivas**. Use apenas em ambientes próprios, labs autorizados ou com permissão explícita. A instalação requer `sudo` e modifica o sistema — confirme antes de prosseguir. ## Licença MIT
