# 🚀 DDfetch A lightweight and customizable system information tool written in Bash, inspired by tools like fastfetch and neofetch. DDfetch displays system information alongside a custom ASCII art in a clean and minimal layout — perfect for terminal aesthetics and r/unixporn setups. --- ## ✨ Features - 🧠 Written in pure Bash (no dependencies) - 🎨 Custom ASCII art support - ⚡ Fast and lightweight - 🖥 Displays system info: - OS - Kernel - Packages - Shell - Window Manager - CPU - RAM - 🔧 Easy to modify and extend --- ## 📸 Preview (Add a screenshot here) --- ## 📦 Installation ```bash git clone https://github.com/SEU-USUARIO/ddfetch.git cd ddfetch chmod +x ddfetch mv ddfetch ~/.local/bin/ ## Instalação e uso Escrito em **Bash puro** — nenhuma dependência além de um terminal Linux/macOS. ```bash git clone https://github.com/Diogo-Damasceno/DDfetch.git cd DDfetch chmod +x ddfetch # roda direto ./ddfetch # ou instala no PATH sudo cp ddfetch /usr/local/bin/ddfetch # depois: ddfetch ``` ## Exemplo ```bash ddfetch # mostra info do sistema + ASCII art ``` Funciona em qualquer shell POSIX. Personalize a ASCII art e as cores editando as variáveis no topo do script.
