README

# 🚀 DDfetch Ferramenta leve e customizável de informações do sistema, escrita em Bash, inspirada em ferramentas como fastfetch e neofetch. O DDfetch exibe informações do sistema junto com uma arte ASCII customizável em um layout limpo e minimalista — perfeito para a estética do terminal e setups de r/unixporn. --- ## ✨ Funcionalidades - 🧠 Escrito em Bash puro (sem dependências) - 🎨 Suporte a arte ASCII customizável - ⚡ Rápido e leve - 🖥 Exibe informações do sistema: - Sistema operacional (OS) - Kernel - Pacotes - Shell - Gerenciador de janelas (WM) - CPU - Memória (RAM) - 🔧 Fácil de modificar e estender --- ## 📸 Pré-visualização _(adicione uma captura de tela aqui)_ --- ## 📦 Instalação e uso Escrito em **Bash puro** — nenhuma dependência além de um terminal Linux/macOS. ```bash git clone https://github.com/Diogo-Damasceno/DDfetch.git cd DDfetch chmod +x ddfetch # roda direto ./ddfetch # ou instala no PATH sudo cp ddfetch /usr/local/bin/ddfetch # depois: ddfetch ``` ## Exemplo ```bash ddfetch # mostra info do sistema + arte ASCII ``` Funciona em qualquer shell POSIX. Personalize a arte ASCII e as cores editando as variáveis no topo do script.

Como rodar na prática (entrada → saída):

$ ddfetch
