# jwt-inspector Inspeciona claims de JWT (leitura, sem quebrar assinatura). > **Aviso etico:** ferramenta exclusiva para fins educacionais e testes em > ambientes que voce possui ou tem autorizacao para auditar. Categoria: > Defensiva. Nunca a use contra terceiros. ## Instalacao ```bash git clone https://github.com/Diogo-Damasceno/jwt-inspector.git cd jwt-inspector python3 -m venv .venv && . .venv/bin/activate pip install -e . ``` ## Uso Revise tokens que sua aplicação emite (exp, escopo, alg). Exemplos reais: ``` $ jwt-inspector eyJhbGc... {"header": {...}, "payload": {...}}``` ## Arquitetura `src/jwt_inspector/core.py` tem a logica pura (funcoes testaveis); `cli.py` e a casca de argumentos. Testes em `tests/` cobrem os casos principais.
